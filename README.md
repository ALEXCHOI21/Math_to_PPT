# Antigravity Math Engine 🚀

### 초격차 PPT 수학 수식 제작 플랫폼

파워포인트(PPT) 내 수학 수식 입력의 한계를 극복하고, 한글(HWP) 수식 복사 시 발생하는 배경색 문제(블랙 박스 현상)를 완벽하게 해결하기 위한 상용 수준의 웹 플랫폼입니다.

## ✨ 주요 특징 (Key Features)

- **Premium UI/UX**: 다크 모드 기반의 세련된 디자인과 글래스모피즘 스타일 적용.
- **Real-time Preview**: LaTeX 입력 시 즉시 고해상도 수식 렌더링 (MathJax 3 적용).
- **Transparent PNG Support**: 투명도가 포함된 PNG 이미지를 클립보드에 바로 복사하여 PPT에 즉시 활용 가능.
- **SVG Vector Download**: 확대해도 깨지지 않는 벡터 그래픽(SVG) 포맷 지원.
- **Formula Presets**: 자주 사용하는 근의 공식, 피타고라스 정리 등 주요 공식 템플릿 제공.

## 🛠 사용 방법 (How to Use)

1. **LaTeX 입력**: 수식 입력창에 원하는 LaTeX 문법으로 수식을 작성합니다.
2. **복사 및 다운로드**:
   - `클립보드 복사 (PNG)`: 즉시 복사하여 PPT에서 `Ctrl + V`로 붙여넣습니다.
   - `SVG 다운로드`: 고품질 벡터 파일이 필요할 때 사용합니다.
3. **PPT 삽입**: 투명 배경이 유지된 상태로 수식이 삽입되는 것을 확인합니다.
