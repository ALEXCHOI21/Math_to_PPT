# Antigravity Math Engine 🚀

### 초격차 PPT 수학 수식 제작 플랫폼 (v1.4.2)

파워포인트(PPT) 내 수학 수식 입력의 한계를 극복하고, 한글(HWP) 수식 복사 시 발생하는 배경색 문제(블랙 박스 현상)를 완벽하게 해결하기 위한 상용 수준의 웹 플랫폼입니다.

## ✨ 주요 특징 (Key Features)

- **Premium UI/UX**: 다크 모드 기반의 세련된 디자인과 글래스모피즘 스타일 적용.
- **Ultra-Fast Rendering**: **KaTeX** 엔진을 탑재하여 수만 개의 수식도 실시간으로 선명하게 렌더링.
- **Intelligent Search**: 50개 이상의 핵심 공식(수능 킬러 문항 포함)을 키워드 하나로 검색하여 즉시 입력.
- **Transparent PNG Support**: 투명 배경 PNG 이미지를 클립보드에 바로 복사하여 PPT에 즉시 활용.
- **SVG Vector Download**: 무한 확대 시에도 깨짐 없는 정교한 벡터 그래픽 지원.
- **Integrated Help Manual**: 플랫폼 내장형 퀵 레퍼런스를 통해 LaTeX 문법을 몰라도 즉시 전문가 수준의 수식 제작 가능.

## 🛠 사용 방법 (How to Use)

1. **LaTeX 입력 또는 검색**: 수식 입력창에 LaTeX를 직접 작성하거나, 우측 라이브러리에서 원하는 공식을 검색하여 클릭합니다.
2. **복사 및 다운로드**:
   - `클립보드 복사 (PNG)`: 즉시 복사하여 PPT에서 `Ctrl + V`로 붙여넣습니다.
   - `SVG 다운로드`: 고품질 벡터 파일이 필요할 때 사용합니다.
3. **PPT 삽입**: 투명 배경이 유지된 상태로 수식이 삽입되는 것을 확인합니다.
