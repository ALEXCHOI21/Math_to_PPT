# [사용자 매뉴얼] Antigravity Math Engine 마스터 가이드

본 지침서는 Antigravity Math Engine을 통해 전문가 수준의 수학 수식을 PPT에 삽입하고자 하는 사용자를 위해 작성되었습니다.

## 1. 기본 사용 프로세스 (3-Step)

1.  **입력**: 상단 'LaTeX 입력' 칸에 원하는 수식 문법을 작성합니다. (우측 라이브러리 프리셋 활용 가능)
2.  **확인**: 하단 '실시간 프리뷰' 영역에서 렌더링된 결과물과 투명도 설정을 확인합니다.
3.  **활용**: `클립보드 복사` 버튼을 눌러 복사한 뒤, PPT에서 `Ctrl + V`로 붙여넣습니다.

---

## 2. LaTeX 퀵 레퍼런스 (주요 문법)

자주 사용되는 문법들을 조합하여 복잡한 수식을 완성할 수 있습니다.

| 구분 | LaTeX 문법 | 결과 예시 |
| :--- | :--- | :--- |
| **분수** | `\frac{분자}{분모}` | $\frac{a}{b}$ |
| **거듭제곱** | `a^n` | $a^n$ |
| **첨자(아래)** | `a_{n}` | $a_{n}$ |
| **루트(근호)** | `\sqrt{x}` | $\sqrt{x}$ |
| **그리스 문자** | `\alpha, \beta, \gamma, \pi` | $\alpha, \beta, \gamma, \pi$ |
| **시그마/합** | `\sum_{i=1}^{n}` | $\sum_{i=1}^{n}$ |
| **적분/인테그랄** | `\int_{a}^{b}` | $\int_{a}^{b}$ |
| **벡터/화살표** | `\vec{F}` | $\vec{F}$ |

> [!TIP]
> **중괄호 `{ }` 활용**: 명령어나 지수에 두 글자 이상의 내용이 들어갈 때는 반드시 `{ }`로 묶어주세요.  
> (예: `e^ix` ➔ `e^{ix}`)

---

## 3. PPT 최상의 품질을 위한 팁

*   **배경 투명도 유지**: 우측 상단의 '투명 배경' 체크박스가 활성화된 상태에서 복사하세요. PPT의 다크 모드나 색상 테마에 구애받지 않고 자연스럽게 삽입됩니다.
*   **고해상도 보장**: 본 플랫폼은 4배 고해상도 렌더링을 지원합니다. 삽입 후 PPT에서 크기를 조절해도 깨지지 않는 선명함을 유지합니다.
*   **벡터(SVG) 활용**: 만약 수식을 무한대로 확대해야 한다면, `SVG 다운로드` 버튼을 통해 다운로드한 파일을 PPT로 드래그 앤 드롭 하세요.

## 4. 자주 묻는 질문 (FAQ)

**Q. 수식이 빨간색 에러 메시지로 나옵니다.**  
A. LaTeX 문법 오타일 확률이 높습니다. 괄호가 닫히지 않았거나 명령어 앞에 백슬래시(`\`)가 누락되지 않았는지 확인하세요.

**Q. 백슬래시(`\`)를 입력했는데 원화(`₩`) 기호로 보입니다.**  
A. 이는 한글 Windows 시스템의 폰트 표시 특징일 뿐입니다. 실제 데이터와 수식 변환에는 문제가 없으니 그대로 `\frac` 등을 사용하시면 됩니다.

**Q. 복사가 되지 않습니다.**  
A. 브라우저의 클립보드 접근 권한이 '차단'되어 있을 수 있습니다. 주소창 옆의 자물쇠 아이콘을 눌러 권한을 허용해 주세요.
