# [사용자 매뉴얼] Antigravity Math Engine 마스터 가이드 (v1.4.1)

본 지침서는 Antigravity Math Engine을 통해 전문가 수준의 수학 수식을 PPT에 삽입하고자 하는 사용자를 위해 작성되었습니다.

## 1. 기본 사용 및 검색 프로세스

1.  **지능형 검색**: 우측 '수식 라이브러리' 상단의 검색창에 **'30번'**, **'근'**, **'Physics'** 등 키워드를 입력하여 원하는 공식을 즉시 찾습니다.
2.  **원클릭 입력**: 검색된 수식 카드를 클릭하면 좌측 'LaTeX 입력' 창에 즉시 복사됩니다.
3.  **실시간 프리뷰**: 하단 영역에서 렌더링 결과를 확인합니다. (오류 시 빨간색 메시지 표시)
4.  **PPT 활용**: `클립보드 복사` 버튼으로 복사한 뒤, PPT에서 `Ctrl + V`로 붙여넣습니다.

---

## 2. LaTeX 퀵 레퍼런스 (주요 및 고급 문법)

기본 문법부터 고난도 수능 문항용 고급 문법까지 요약되어 있습니다.

| 구분 | LaTeX 문법 | 결과 예시 |
| :--- | :--- | :--- |
| **자동 크기 괄호** | `\left( \frac{a}{b} \right)` | $\left( \frac{a}{b} \right)$ (괄호 크기 자동 조절) |
| **조건부 함수** | `\begin{cases} x & x \ge 0 \\ -x & x < 0 \end{cases}` | $\begin{cases} x & x \ge 0 \\ -x & x < 0 \end{cases}$ |
| **분수** | `\frac{분자}{분모}` | $\frac{a}{b}$ |
| **루트(근호)** | `\sqrt{x}` | $\sqrt{x}$ |
| **그리스 문자** | `\pi, \alpha, \theta` | $\pi, \alpha, \theta$ |
| **극한(Limit)** | `\lim_{x \to \infty}` | $\lim_{x \to \infty}$ |
| **시그마/합** | `\sum_{i=1}^{n}` | $\sum_{i=1}^{n}$ |

> [!TIP]
> **수능 킬러 문항 팁**: 복잡한 수식일수록 `\left(` 와 `\right)`를 적절히 사용하면 가독성이 비약적으로 상승합니다. 

---

## 3. PPT 최상의 품질을 위한 팁

*   **배경 투명도 유지**: '투명 배경' 체크박스가 활성화된 상태에서 복사하여 PPT 테마와 자연스럽게 어우러지게 하세요.
*   **초고해상도**: 본 플랫폼은 4~5배 고해상도 캡처를 지원하여, PPT 내부에서 이미지를 키워도 계단 현상 없이 선명합니다.
*   **SVG 벡터 활용**: 수식을 무한대로 확대해야 할 경우 `SVG 다운로드` 후 PPT로 드래그 앤 드롭 하세요.

## 4. 자주 묻는 질문 (FAQ)

**Q. 백슬래시(`\`)가 원화(`₩`) 기호로 보입니다.**  
A. 한글 윈도우 시스템의 폰트 표시 특징일 뿐입니다. 실제 수식 변환에는 문제가 없으니 그대로 사용하십시오.

**Q. 수식이 빨간색으로 나옵니다.**  
A. 문법 오류입니다. 특히 `\begin{cases}`를 썼다면 반드시 `\end{cases}`로 닫아주었는지 확인하세요.
